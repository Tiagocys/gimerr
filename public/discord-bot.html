App do Gimerr no Discord

Conecte o Gimerr a canais de texto do seu servidor para publicar automaticamente conteúdo de jogos específicos.

Como funciona

Quando alguém publicar no Gimerr um vídeo ou anúncio de um jogo que você escolheu, o App envia automaticamente uma mensagem no canal-de-texto escolhido, siga o passo a passo abaixo para configurar o app Gimerr no seu Discord.

1

Entre no Gimerr com Discord

Use a mesma conta do Discord que você usa para administrar o servidor. Se a conta já estiver conectada ao seu perfil, siga para a instalação.

Entrar com Discord
2

Instale o App no servidor

Autorize o App do Gimerr no Discord. Ele precisa ver canais, enviar mensagens e inserir links para publicar os cards dos anúncios.

Instalar App no servidor
3

Configure o canal

No canal de texto que receberá os anúncios, use o comando abaixo para escolher o game e o tipo de conteúdo das atualizações:

/gimerr setup game
4

Confira ou remova

Use os comandos abaixo para revisar o que está ativo no canal ou remover uma configuração.

/gimerr status /gimerr remove game
